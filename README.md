# JSON Viewer

> Paste, drop, or browse a JSON file — get an interactive, syntax-highlighted tree instantly.

Built with React 18 + TypeScript + Vite. No accounts, no server, no fuss.

---

## Getting Started

```bash
npm install
npm run dev        # → http://localhost:3000
```

```bash
npm run build      # production bundle → dist/
```

---

## What It Does

| Feature | Detail |
|---------|--------|
| **Interactive tree** | Expand/collapse nodes, copy to clipboard |
| **Multiple inputs** | Type/paste JSON, drag-drop a file, or paste from clipboard |
| **Real-time validation** | Instant error messages as you type |
| **Format** | One-click JSON pretty-print |
| **Dark / Light theme** | Toggle anytime, smooth transition |
| **Keyboard friendly** | Full keyboard nav + ARIA labels |

---

## Stack

- **React 18** + **TypeScript** (strict)
- **Vite 5** — fast dev server & build
- **CSS Modules** — scoped styles, no Tailwind
- **@textea/json-viewer** — JSON tree rendering
- **Fira Code** — monospace font
